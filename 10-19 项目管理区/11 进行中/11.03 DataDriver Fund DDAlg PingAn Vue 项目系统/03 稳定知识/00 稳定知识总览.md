## 当前已确认的稳定知识
- 项目是 Vue 2 + Element UI + VXE Table + BPMN-js + AntV X6 + Webpack 4 的老项目体系。
- 这是一个业务范围较宽的指标平台前端，既覆盖指标配置，也覆盖报表、产品、流程和数据视图能力。
- 真实入口判断不能只看文件名，必须警惕 `old`、`copy`、`backup` 等历史遗留目录。
- 最重要的定位信息不是抽象架构，而是“中文业务名 -> 页面路径 -> API 文件”的快速映射。
- 图形化相关问题优先从业务页面反查 `src/components/WorkflowChart`、`src/components/algGraph` 等组件。
