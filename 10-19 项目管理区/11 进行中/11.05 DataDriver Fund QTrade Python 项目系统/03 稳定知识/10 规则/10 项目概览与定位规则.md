# 项目概览与定位规则

## 项目定位
- QTrade Python 当前识别为 DataDriver Fund 量化交易 / 选股 / 缠论分析相关 Python 后端项目。
- 代码主体位于 `backend`，包含 Django 应用、REST API、Celery、系统管理、选股分析、缠论核心和 czsc 相关代码。

## 稳定入口
- Django 项目入口：`backend/manage.py`
- 配置入口：`backend/application/settings.py`
- 路由入口：`backend/application/urls.py`
- 系统管理：`backend/dvadmin/system`
- 选股业务：`backend/dvadmin/selection`
- 缠论核心：`backend/core/chanlun`
- czsc 代码：`backend/czsc`

## 协作规则
- 修改前按任务读取最小必要上下文。
- 先分清业务代码、框架代码、迁入第三方库代码边界。
- 涉及量化逻辑时，优先补充可复现输入、预期输出和回归验证。
