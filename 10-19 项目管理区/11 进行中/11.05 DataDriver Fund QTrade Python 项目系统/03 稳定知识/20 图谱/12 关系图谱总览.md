- `backend/manage.py` -> `backend/application/settings.py` / `backend/application/urls.py`
- `backend/application/urls.py` -> `api/system/` -> `backend/dvadmin/system/urls.py` -> 系统管理 views / models / utils
- `backend/application/urls.py` -> `api/selection/` -> `backend/dvadmin/selection/urls.py` -> 选股、回测、缠论、TradingView views
- `backend/dvadmin/selection/views` -> `backend/dvadmin/selection/services` / `task_services` -> `backend/core/chanlun` / `backend/czsc`
- `backend/application/celery.py` -> `backend/dvadmin/selection/tasks.py` / `task_services` -> Redis / MySQL / 行情数据更新
