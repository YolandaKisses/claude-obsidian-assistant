# DataDriver Fund DDAlg PingAn Vue

## 项目信息
- **路径:** `/Users/yolanda/Documents/WorkSpace工作区/指标中心/datadriver-fund-ddalg-pingan-vue`
- **技术栈:** Vue 2 + Element UI + VXE Table + BPMN-js + AntV X6
- **状态:** 活跃开发中
- **架构图:** [[DataDriver Fund DDAlg PingAn Vue Architecture.canvas]]

## 技术栈表格
| Layer | Tech | Purpose |
|-------|------|---------|
| 框架 | Vue 2.5 | 前端框架 |
| UI组件 | Element UI 2.12 | 基础UI组件库 |
| 表格组件 | VXE Table 2.11 | 高性能表格组件 |
| 流程设计 | BPMN-js 2.4 | BPMN流程图设计 |
| 可视化 | AntV X6 2.18 | 图形编辑与关系图 |
| 图表 | ECharts 5.4 | 数据可视化 |
| 状态管理 | Vuex 3.0 | 全局状态管理 |
| 路由 | Vue Router 3.0 | 路由管理 |
| 构建工具 | Webpack 4.16 | 项目构建 |

## 核心模块
1. **重分类管理** - 指标重分类配置、树形结构管理、Excel导入导出
2. **产品配置管理** - 基金产品配置、明细表分页、批量操作
3. **流程管理** - BPMN流程图设计、流程审批配置
4. **可视化** - X6图形编辑、指标关系可视化
