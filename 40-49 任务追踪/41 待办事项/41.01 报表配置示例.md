# ReportRenderer 通用报表组件

  

## 使用方式

  

```vue

<template>

<report-renderer

report-code="报表编码"

:init-params="初始参数"

:external-columns="外部列配置"

:external-data="外部数据"

/>

</template>

  

<script>

import ReportRenderer from './ReportRenderer'

  

export default {

components: { ReportRenderer }

}

</script>

```

  

## JSON 配置格式

  

```json

{

"tableName": "报表标题",

"sqlCode": "sqlCode编码",

"isPage": 1,

"searchFields": [],

"fields": []

}

```

  

## 顶层配置

  

| 字段 | 类型 | 说明 |

|------|------|------|

| `tableName` | string | 表格标题 |

| `sqlCode` | string | 查询的 sqlCode 编码（用于 `indexCode` 参数） |

| `isPage` | number | `1`=分页, `0`=不分页 |

| `searchFields` | array | 查询表单字段配置 |

| `fields` | array | 表格列配置 |

  

## 查询表单配置 (searchFields)

  

| 字段 | 类型 | 说明 |

|------|------|------|

| `fieldEn` | string | 字段英文名（参数名） |

| `fieldCn` | string | 字段中文名（表单标签） |

| `fieldType` | string | `C`=字符, `N`=数字, `D`=日期 |

| `inputType` | string | 输入类型，见下方 |

| `dimType` | string | 下拉字典类型（如 `QSY300`） |

| `formatValue` | string | 日期格式：`yyyy`/`yyyy-mm`/`yyyy-MM-dd`/`range` |

| `defaultValue` | string | 默认值 |

| `required` | boolean | 是否必填 |

| `sonEn` | string | 联动子字段名 |

  

### 输入类型 (inputType)

  

| 值 | 组件 | 说明 |

|------|------|------|

| `text` | el-input | 普通输入框 |

| `date` | el-date-picker | 日期选择 |

| `select` | el-select | 单选下拉 |

| `select_multiple` | el-select (multiple) | 多选下拉 |

| `select_fund` | 基金选择组件 | 单选基金 |

| `select_multiple_fund` | 基金多选组件 | 多选基金 |

| `selectTable` | 动态报表下拉 | 根据另一报表获取选项 |

  

## 表格列配置 (fields)

  

### 基础属性

  

| 字段 | 类型 | 说明 |

|------|------|------|

| `fieldEn` | string | 字段英文名（唯一标识） |

| `fieldCn` | string | 字段中文名（列表表头） |

| `fieldType` | string | 字段类型，见下方 |

| `width` | number | 列宽（px） |

| `align` | string | 对齐方式：`left` / `center` / `right` |

| `format` | string | 数字格式化 |

| `dateFormat` | string | 日期格式化 |

| `merge` | boolean | 是否启用行合并 |

| `colorRules` | array | 颜色规则 |

| `defaultValue` | string | 查询表单默认值 |

| `required` | boolean | 是否必填 |

  

### 字段类型 (fieldType)

  

| 值 | 含义 |

|------|------|

| `C` | 字符/文本 |

| `N` | 数字 |

| `D` | 日期 |

| `G` | 分组表头（用于表头合并，有 children） |

  

## 表头合并分组 (children)

  

通过 `fieldType: 'G'` 定义分组，配合 `children` 放置子列：

  

```json

{

"fieldEn": "fundInfo",

"fieldCn": "基金信息",

"fieldType": "G",

"width": 360,

"children": [

{ "fieldEn": "rowNo", "fieldCn": "序号", "fieldType": "N", "width": 60 },

{ "fieldEn": "fundCode", "fieldCn": "基金代码", "fieldType": "C", "width": 100 },

{ "fieldEn": "fundName", "fieldCn": "基金名称", "fieldType": "C", "width": 200 }

]

}

```

  

效果：

  

```

| 基金信息 |

| 序号 | 基金代码 | 基金名称 |

```

  

## 行合并 (merge)

  

配置 `merge: true`，相邻行相同值自动合并：

  

```json

{

"fieldEn": "fundName",

"fieldCn": "基金名称",

"merge": true

}

```

  

## 数字格式化 (format)

  

| 格式 | 说明 | 示例 |

|------|------|------|

| `#,##0.00` | 千分位，保留2位小数 | 1,234.56 |

| `0.00%` | 百分比（小数转百分比） | 23.45% |

| `#,##0.000` | 千分位，保留3位小数 | 1,234.567 |

| `#,##0.0000` | 千分位，保留4位小数 | 1.2345 |

  

示例：

```json

{ "fieldEn": "nav", "fieldType": "N", "format": "#,##0.0000" }

{ "fieldEn": "return", "fieldType": "N", "format": "0.00%" }

```

  

## 颜色规则 (colorRules)

  

根据字段值动态设置字体颜色：

  

```json

{

"colorRules": [

{ "operator": ">", "value": 0, "color": "green" },

{ "operator": "<", "value": 0, "color": "red" }

]

}

```

  

**支持的操作符：**

  

| 操作符 | 含义 |

|--------|------|

| `>` | 大于 |

| `>=` | 大于等于 |

| `<` | 小于 |

| `<=` | 小于等于 |

| `==` | 等于 |

| `!=` | 不等于 |

  

**颜色值：** 支持 CSS 颜色名称（`green`, `red`, `orange`）或十六进制（`#52c41a`, `#f5222d`）

  

**示例 - 涨跌幅：**

```json

{

"fieldEn": "dailyReturn",

"fieldCn": "日涨幅",

"fieldType": "N",

"colorRules": [

{ "operator": ">", "value": 0, "color": "#52c41a" },

{ "operator": "<", "value": 0, "color": "#f5222d" }

]

}

```

## 接口格式

  

### 查询数据（通用 sqlCode 接口）

  

**请求：** `POST /api/common/getDataBySqlCode`

  

```json

{

"indexCode": "2d0c66c5-b4b0-43fa-b784-ad4d545dd57e",

"sqlCode": "2d0c66c5-b4b0-43fa-b784-ad4d545dd57e",

"pageNum": 1,

"pageSize": 20,

"orderString": "fundCode asc",

"fundCode": "000001",

"fundName": "华夏成长混合"

}

```

  

**说明：**

- `indexCode` / `sqlCode`：页面配置的 sqlCode 编码

- `pageNum` / `pageSize`：分页参数

- `orderString`：排序字符串

- 其他参数：来自 searchFields 表单字段

  

**响应：**

```json

{

"data": {

"status": 0,

"data": [

{ "fundCode": "000001", "fundName": "华夏成长混合", ... }

]

}

}

```

  

### 导出报表

  

**请求：** `POST /api/reportConfig/v1.0/exportReport`

  

```json

{

"filterParams": "{\"fundCode\":\"000001\"}",

"tableHeader": ["fundCode", "fundName", "nav"],

"token": "xxx",

"type": "excel"

}

```

  

## 组件属性

  

| 属性 | 类型 | 说明 |

|------|------|------|

| `report-code` | String | 报表编码（必填） |

| `init-params` | Object | 初始查询参数 |

| `external-columns` | Array | 外部列配置（优先级高于接口） |

| `external-data` | Array | 外部数据（优先级高于接口） |
